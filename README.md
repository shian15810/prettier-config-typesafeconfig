# prettier-config-typesafeconfig

Prettier base configuration (prettierrc) to extend from with emphasis on type safety.

## Installation

```shell
npm install --save-dev prettier-config-typesafeconfig
```

## Usage

In your `.prettierrc.json`:

```json
"prettier-config-typesafeconfig"
```
